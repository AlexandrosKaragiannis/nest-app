# nest-app

## Description
Test OpenAPI spec on NestJS backend

## Installation
Clone this repo and run
```bash
$ npm install
```

## Running the app
To start the app run
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

For now, the api routes can be viewed at
`http://localhost:3000/api/`
